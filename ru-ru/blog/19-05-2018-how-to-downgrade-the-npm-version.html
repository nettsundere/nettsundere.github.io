Как откатить версию NPM (менеджера пакетов node).

Наберите в консоли:

npm install -g npm@3.10.10 (в этом примере версия — 3.10.10)

Изначально опубликовано на Medium: https://medium.com/@nettsundere/how-to-downgrade-the-npm-node-package-manager-version-c9650af30ba6
